linux commands 
Символы:
ctr+alt+t- команда открытия терминала 
/- символ корневой дериктории
~-символ домашнец дериктории
. - обозначает текущую директорию
.. - обозначает предыдущую директорию

Команды:
ls- команда для вывода всех дерикторий(папка) и файлов в текущей директории в терминале 
mkdir - создать директорий
rmdir - удалить директорий
cd (change directory) - команда для перехода из одной директории в другую
touch - создает файл
rm - ужаляет файл
rm -rf --удаляет вообще все(принудительное удаление)
ls -a - показывает скрытые и обычные файлы и директории
ls -la - показывает подробную инфу
pwd(print working directory) - выводит путь до текущей директории 
cp - копирует файл и директорию в указанное место (cp <название><куда>)
mv - команда для перемещения файла и дериктории а так же для переименовывания файла и директории
переименовывание mv <old name> <new name>
перемешение mv <name> <directory/>
cat - показывает содержание файла
clear - очищение
nano - открывает файл для редактирования в графическом интрефейсе Нано * если нет файла, то нано создаст новый
echo - команда для быстрой записи текста в файл 
echo "text" > file
